Commands - Vosk Edit

Vosk-Edit:

Vosk ist ein Tool, welches dazu dient, sprache zu erkennen und
aufzuzeichnen.

Commands:

Generell fangen alle Befehle mit dem Codewort "Brot"
Versuchen Sie bitte, so deutlich wie möglich zu Sprechen!
Das Programm gibt Ihnen aus, welche wörter es erkennen kann

Buchstaben

Buchstabe a: "brot a"
Buchstabe b: "brot b"
...

Alternativ:
Buchstabe a: "brot buchstabe a"
Buchstabe b: "brot buchstabe b"
...
